#  Quiz: Que Tipo de Programador Você Seria?

##  Sobre o projeto
Este projeto é um **quiz interativo sobre tecnologia** que descobre qual área da programação combina mais com o usuário.

Ao responder algumas perguntas, o sistema analisa as respostas e mostra um resultado explicando qual tipo de programador mais se aproxima do perfil da pessoa.

O projeto foi desenvolvido utilizando **HTML, CSS e PHP**.

---

##  Objetivo
O objetivo do quiz é:
- Apresentar diferentes áreas da programação
- Tornar o aprendizado sobre tecnologia mais divertido
- Permitir que o usuário descubra qual área da programação combina mais com seus interesses
